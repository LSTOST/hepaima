本文档定义了「我们合拍吗」产品的完整视觉设计系统，确保所有页面风格统一。

## 🎨 设计基础

### 品牌调性

| 关键词 | 说明 | 体现方式 |
|--------|------|----------|
| **温暖** | 让人感到亲近舒适 | 粉紫色系、圆润造型 |
| **专业** | 值得信赖的心理学产品 | 清晰排版、克制装饰 |
| **现代** | 符合年轻人审美 | 简洁布局、精致细节 |
| **亲密** | 契合情侣产品属性 | 柔和色彩、温馨文案 |

## 🧩 通用组件

### 按钮 Button

```
主要按钮 Primary：
  背景：#111111
  文字：#FFFFFF
  圆角：8px
  内边距：12px 24px（常规）/ 8px 16px（小）/ 14px 32px（大）
  hover：背景变为 #333333

强调按钮 Accent：
  背景：渐变 #EC4899 → #8B5CF6
  文字：#FFFFFF
  hover：轻微放大 scale(1.02)

次要按钮 Secondary：
  背景：#F5F5F5
  文字：#333333
  边框：1px solid #DDDDDD
  hover：背景 #EEEEEE

幽灵按钮 Ghost：
  背景：透明
  文字：#666666
  hover：背景 #F5F5F5
```

### 卡片 Card

```
基础卡片：
  背景：#FFFFFF
  边框：1px solid #EAEAEA
  圆角：12px
  内边距：16px - 24px
  
hover状态：
  边框：1px solid #DDDDDD
  阴影：0 8px 24px rgba(0,0,0,0.08)
  位移：translateY(-2px) 或 translateY(-4px)

强调卡片（如热恋期）：
  边框：1px solid #EC4899
  可选背景：linear-gradient(135deg, #FDF2F8, #FFFFFF)
```

### 输入框 Input

```
默认状态：
  背景：#FFFFFF
  边框：1px solid #DDDDDD
  圆角：8px
  内边距：12px 16px

聚焦状态：
  边框：1px solid #EC4899
  阴影：0 0 0 3px rgba(236, 72, 153, 0.1)

错误状态：
  边框：1px solid #EF4444

```

#### 设计要点

**导航栏**
- 左：Logo（可简化为文字）
- 中：当前阶段标签（用Badge样式）
- 右：进度指示（如 5/32）

**进度条**
- 使用渐变色进度条
- 下方可显示百分比

**题目**
- 题号 + 题目文字
- 字号：18-20px，字重 Medium
- 颜色：#111111

**选项卡片**
- 每个选项是一个可点击卡片
- 左侧：选项字母（A/B/C/D），用圆角方块包裹
- 右侧：选项文字
- 默认：白底 + 浅灰边框
- Hover：边框加深
- 选中：边框变主色 + 背景变浅粉

**交互**
- 点击选项后，短暂延迟（0.3s）自动进入下一题
- 提供"上一题"按钮（Ghost样式）
- 最后一题点击后进入提交状态

**加载/提交状态**
- 显示loading动画
- 文案："正在生成报告..." 或 "AI正在分析你们的答案..."

## 📱 响应式规范

### 断点

```
移动端：< 640px（主要适配目标）
平板端：640px - 1024px
桌面端：> 1024px
```

### 移动端优先原则

- 所有设计先考虑移动端
- 桌面端是"放大"而非"缩小"
- 触摸目标最小 44px
- 避免横向滚动

---

## ✨ 动效规范

### 过渡时长

```
快速：150ms  - 按钮状态变化
常规：200ms  - 卡片hover
中等：300ms  - 页面元素进入
慢速：500ms  - 模态框出现
```

### 缓动函数

```
默认：ease
弹性：cubic-bezier(0.34, 1.56, 0.64, 1)
```

### 常用动效

- 按钮hover：背景色变化 + 轻微上移
- 卡片hover：阴影加深 + 上移
- 选项选中：边框/背景变化 + 缩放微动
- 页面切换：淡入淡出或滑动
- 加载状态：骨架屏或spinner
